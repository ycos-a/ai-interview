## 项目简介

面面-AI求职助手是一个利用大语言模型（LLM）和 LangGraph 状态机技术构建的综合求职辅助系统。它不仅能进行全真模拟面试，还能像专业的职业咨询师一样，通过多智能体协作（Multi-Agent）对简历进行深度诊断和定向优化，帮助求职者全方位提升竞争力。

**在线体验**: https://interview.1624899.xyz

## 快速开始

### 第一步：访问应用
打开浏览器访问部署好的网站，进入首页。

### 第二步：配置 API
1. 点击右上角**设置图标**
2. 选择你的模型提供商（如 OpenAI、阿里云百炼等）
3. 输入 API Key
4. 点击**测试连接**确认配置成功
5. 点击保存

### 第三步：开始使用

**面试模式**：
1. 点击左侧导航的 **"+"** 创建新会话
2. 上传你的简历（PDF 格式）
3. 输入目标职位描述（JD）
4. 可选：添加公司信息
5. 设置题目数量（3-10道）
6. 选择面试模式（文字/语音）
7. 点击**开始面试**

**简历工具模式**：
1. 点击左侧导航的 **"简历工具"** 标签
2. 选择 **竞争力分析** 或 **内容优化**
3. 粘贴简历内容或上传文件
4. 输入目标职位描述（优化必需）
5. 点击相应按钮开始

## 面试功能使用指南

### 4.1 创建新面试

1. **点击新建会话**
   - 在左侧边栏点击 **"+"** 按钮
   - 系统会跳转到面试配置页面

2. **上传简历**
   - 点击上传区域，选择 PDF 文件
   - 支持文件格式：`.pdf`
   - 上传成功后显示文件名和"已上传"标记

3. **输入职位描述**
   - 点击职位描述区域
   - 在弹窗中详细输入：
     - 职位名称
     - 技术要求
     - 工作年限要求
     - 核心职责等
   - 越详细，生成的面试问题越精准

4. **设置题目数量**
   - 拖动滑块选择 3-10 道题目
   - 建议选择 5 道，既能充分展示又不至于太累

5. **开始面试**
   - 点击底部**"开始面试"**按钮
   - AI 面试官会自动生成第一个问题

### 4.2 面试进行中

**回答问题**：
- 在底部输入框输入你的回答
- 按 Enter 或点击发送按钮提交
- 回答会被发送给 AI 面试官

**获取提示**：
- 如果遇到困难，点击输入框上方的💡灯泡图标
- 系统会生成回答提示供参考

**语音输入**：
- 点击输入框右侧的话筒图标
- 开始语音输入（需要浏览器授权麦克风）

**中断面试**：
- 点击"停止"按钮可以暂停面试
- 面试进度会被保存

### 4.3 查看能力画像

面试结束后：
1. 系统显示**"面试已完成"**提示
2. 点击**"本轮能力画像"**查看详细报告
3. 报告中包含：
   - 各维度能力评分（雷达图）
   - 优势分析
   - 待改进建议
   - 面试洞察

### 4.4 多轮面试

- 一面完成后可点击**"开启下一轮"**
- 简历和 JD 自动继承
- 每一轮面试的问题不会与上一轮重复
- 建议完成 3 轮面试获得更全面的评估

## 简历工具使用指南

### 5.1 竞争力分析

**功能**：对简历进行多维度评估

**操作步骤**：
1. 点击左侧 **"简历工具"** 标签
2. 选择 **"竞争力分析"** 标签页
3. 输入简历内容（粘贴文字或上传文件）
4. 可选：输入目标职位描述
5. 可选：关联之前的面试记录（最多3条）
6. 点击**"开始竞争力分析"**
7. 查看分析结果：
   - 综合评分
   - 各维度评分（结构、完整度、量化、清晰、亮点、JD匹配）
   - 优势分析
   - 待改进建议
   - 智能优化建议（P1/P2优先级）

### 5.2 内容优化

**功能**：针对目标职位优化简历内容

**操作步骤**：
1. 点击左侧 **"简历工具"** 标签
2. 选择 **"内容优化"** 标签页
3. 输入简历内容
4. **必须**输入目标职位描述（JD）
5. 可选：关联面试记录获取更精准建议
6. 可选：开启"包含综合能力画像"获取面试表现加成
7. 点击**"生成内容优化建议"**
8. 查看优化结果：
   - JD 匹配度评分
   - HR 通过率预估
   - 关键改进点
   - 关键词分析（缺失/已匹配）
   - 面试洞察

### 5.3 生成简历

在内容优化结果页面：
1. 查看优化建议后
2. 点击**"下滑直接生成"**或**"生成优化简历"**按钮
3. 在生成对话框中确认信息
4. 等待 AI 生成新简历
5. 预览并编辑生成的简历
6. 支持复制和下载

### 5.4 图片简历识别

系统支持从图片中提取简历内容：
1. 点击**"导入简历"**按钮
2. 选择图片文件（支持 .jpg、.png 格式）
3. 系统自动 OCR 识别提取文字
4. 确认识别结果并编辑

## 常见问题

### Q1: 上传简历失败？
- 确保文件格式为 PDF、DOCX 或 TXT
- 文件大小不超过 10MB
- 如果是图片格式，系统会自动 OCR 识别

### Q2: 面试无法开始？
- 检查是否已配置 API Key
- 确认 API 连接测试通过
- 检查简历内容是否为空

### Q3: AI 回复很慢？
- 检查网络连接
- 可能是 API 提供商服务器繁忙
- 尝试减少题目数量

### Q4: 如何切换语音面试？
- 需要先在设置中配置语音模型（如 Qwen3-Omni）
- 面试过程中可点击"切换语音面试"按钮
- 或在开始面试时选择"语音对话"模式

### Q5: 如何查看历史记录？
- 左侧边栏显示所有会话列表
- 点击任意会话可继续或回顾
- 简历工具的历史记录在右侧面板

### Q6: API 配置建议？

| 使用场景 | 推荐模型 |
|----------|----------|
| 简历分析 | GPT-4 / DeepSeek / Qwen3 |
| 面试对话 | GPT-3.5-Turbo / 快速模型 |
| 语音交互 | Qwen3-Omni（全模态） |

**注意**：简历工具建议使用非免费 API 以确保稳定性
