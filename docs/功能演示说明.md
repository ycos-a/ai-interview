### 面试配置流程

**截图参考**: `Screenshot/1.png`

1. **上传简历**
   - 点击虚线框区域
   - 选择 PDF 文件
   - 显示"已上传"状态

2. **输入职位描述**
   - 点击职位描述卡片
   - 在弹窗中详细填写
   - 建议包含：职位名、技术栈、年限要求

3. **设置参数**
   - 题目数量：3-10 道可调
   - 面试模式：文字/语音

4. **开始面试**
   - 点击深色按钮启动

### 面试进行界面

**截图参考**: `Screenshot/2.png`

- **顶部进度条**：显示当前问题进度

- **输入区域**：文字输入、语音输入、提示按钮

## 简历工具演示

### 简历工具入口

在左侧导航点击 **"简历工具"** 标签进入。

### 竞争力分析

**截图参考**: `Screenshot/4.png`

分析维度：
- 结构规范性
- 内容完整度
- 量化程度
- 表达清晰度
- 亮点突出度
- JD 匹配度

输出内容：
- 综合评分（0-100）
- 各维度评分条
- 优势列表（绿色卡片）
- 待改进列表（橙色卡片）
- 智能优化建议（P1/P2 优先级）

### 内容优化

优化流程：
1. 输入简历内容
2. 输入目标 JD（必需）
3. 可选：关联面试记录
4. 点击"生成内容优化建议"

输出内容：
- JD 匹配度百分比
- HR 通过率预估
- 关键改进点列表
- 关键词分析（缺失/已匹配）
- 可点击生成优化简历

### 简历生成

在优化结果页面：
1. 查看详细优化建议
2. 点击"生成优化简历"按钮
3. 等待 AI 生成
4. 预览和编辑
5. 支持复制和下载

## 功能亮点

### 1. 智能面试规划
- 根据简历和 JD 自动生成个性化问题
- 问题类型涵盖：自我介绍、技术问题、行为面试
- 模拟真实面试体验

### 2. 多轮面试系统
- 支持连续多轮面试
- 问题自动去重
- 简历/JD 自动继承

### 3. 多专家简历诊断
- **匹配分析师**：分析 JD 匹配度
- **内容优化师**：提供修改建议
- **HR 审核官**：模拟筛选视角
- **质量审核**：二次审核确保质量

### 4. 智能能力评估
- 六维度能力雷达图
- STAR 法则拆解
- 综合能力趋势追踪

### 5. 灵活的 API 配置
- 支持 OpenAI、DeepSeek、阿里云百炼等
- 前端动态配置
- 流式响应体验
